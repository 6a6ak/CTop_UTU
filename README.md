# ctop Installer Script

Simple Bash script to install `ctop` (`docker-ctop`) on Ubuntu/Debian systems.

## What is ctop?

`ctop` is a lightweight terminal UI for monitoring Docker containers in real time.

It shows:

- CPU usage
- Memory usage
- Network activity
- Container status
- Logs and process information

Similar to `htop`, but for Docker containers.

---

## Requirements

- Ubuntu / Debian based system
- Docker installed
- sudo privileges
- Internet connection

---

## Installation

Clone the repository or download the script.

Make the script executable:

```bash
chmod +x install-ctop.sh
```
